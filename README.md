# Design-Pattern-Study
유니티(Unity)를 활용하여 **소프트웨어 디자인 패턴을 학습하고 직접 구현해보는 개인 학습용 레포지토리**입니다.
이 레포지토리는 이론 중심이 아닌, **실제 Unity 프로젝트에서의 활용과 구조 이해**를 목표로 합니다.

## 목적
- 대표적인 소프트웨어 디자인 패턴의 개념 이해
- Unity 환경에서 디자인 패턴이 어떻게 적용되는지 학습
- 패턴별 장단점과 사용 시점을 정리
- 게임 개발 관점에서의 실용적인 구조 설계 연습
